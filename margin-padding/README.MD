# Отступы в CSS
Изучаем отступы в CSS. Отступы бывают внешние `margin` и внутренние `padding`.

    отступ: верх (слева/справа) низ;
    margin: 5px 10px 3px;

## margin
    
    // горизонтальное центрирование
    max-width: 100px;
    margin: 0 auto;

    // прикрепить блок справа
    max-width: 300px;
    margin-left: auto;
